FORMAT

#fileName.type          Nessisary
:Image name
-Image description
?Note 1
?Note 2
!                       End of data Nessisary



RULES

All photos must have an entry or it will not display
All photos must have "#FileName" for the photo to display
":" "-" "?" do not need values and will display as NONE
Order of ":" "-" "?" does not matter
"#" MUST be at the start of each entry, tells the script where to begin
"!" MUST be at the end of each entry, it tell the script where that entry ends



EXAMPLES

#1234567890
:Nice sunset
-Picture of a nice sunset
?Took this photo as a commision for stock photo website
?The beach in the image is Bondi
!

#2345678901
!
